Multi-Transport & Session Management

Built-in support for HTTP (WebBroker, Indy) and STDIO transports. Thread-safe stateful sessions are injected automatically via attributes, with configurable timeout, automatic cleanup, and support for both generic JSON and fully typed custom session classes.
